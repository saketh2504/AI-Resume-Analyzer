# AI Resume Analyzer

AI Resume Analyzer is a full-stack web application that uses Artificial Intelligence to analyze resumes, compare them with job descriptions, calculate ATS compatibility scores, identify missing skills, and generate AI-powered interview questions.

---

## 🚀 Live Demo

**Frontend:** https://ai-resume-analyzer-blond-iota.vercel.app

**Backend:**  https://ai-resume-analyzer-h3ge.onrender.com

## ⚙️ Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
