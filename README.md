# AI Resume Analyzer

AI Resume Analyzer is a full-stack web application that uses Artificial Intelligence to analyze resumes, compare them with job descriptions, calculate ATS compatibility scores, identify missing skills, and generate AI-powered interview questions.

---

## Features

- Resume Upload (PDF)
- Resume Parsing
- ATS Score Prediction
- Job Description Matching
- Missing Skills Detection
- AI Resume Suggestions
- AI Interview Question Generation
- User Authentication
- Responsive UI

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
